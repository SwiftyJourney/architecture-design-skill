## Getting Started

1. Fork the repository
2. Clone your fork
3. Create a branch: `git checkout -b feature/your-feature`
4. Make changes
5. Test locally
6. Submit PR

---

## Adding Examples

To add new examples (Python, Kotlin, TypeScript):

1. Create directory: `examples/python/`
2. Add markdown files with clear examples
3. Follow existing format
4. Reference Essential Developer principles

---

## Code Style

- Use ATX-style headers (`#`)
- Code blocks with language tags
- Clear, commented examples
- Follow Conventional Commits

---

## Testing

## Pull Requests
